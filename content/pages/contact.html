Community

You building a great app using Buddycloud helps grow the whole Buddycloud community. Please reach out if you get stuck: we want to help you build a successful application.


How can I contribute to the open source project?

The Buddycloud community contribute fixes to the open source code and help design future messaging products.

The best place to start is introducing yourself to other Buddycloud developers. We love pull requests: each project gets tagged with "Help Wanted" (for example: buddycloud-server).

You can also drop by the Buddycloud developer chat room and get direct support from other developers building with Buddycloud.
